# Compact Keywords & Sitelinks

How we use **compact, informative** titles and link text so Google (and users) can show and understand our pages at a glance—including sitelinks under search results.

---

## What are “compact keywords”?

**Compact keywords** (in an SEO/sitelinks context) are **short, descriptive phrases** used for:

1. **Page titles** and **meta descriptions** (e.g. &lt; 60 characters for the main title)
2. **Navigation and link labels** (e.g. “Angel Numbers”, “Life Path Calculator”)
3. **Headlines and subheads** when others (press, partners) reference us

Google’s guidance for [sitelinks](https://developers.google.com/search/docs/appearance/sitelinks) says: *“Use compact, informative page titles and headings”* and *“Use concise, relevant anchor text for internal links.”* Sitelinks are the extra links that appear under your main result in search; clear, short labels help Google choose good ones and help users know where they’re going.

So “compact keywords” here means: **short, high-intent phrases** we use consistently for our key pages and when describing Spirit Numeral (including in the press kit).

---

## How we apply them

### 1. Navigation & footer

We use short, consistent labels in the navbar and footer so both users and crawlers see the same phrases:

- **Angel Numbers** → `/meaning/angel-number`
- **Life Path Numbers** → `/meaning/life-path`
- **Life Path Calculator** → `/calculator`
- **Number Quiz** or **Angel Number Quiz** → `/quiz`
- **Compare Numbers** → `/compare`
- **Compatibility** → `/compatibility`
- **Blog** → `/blog`

Avoid long or vague labels (e.g. “Discover the meaning of angel numbers” in the nav); keep nav items compact and scannable.

### 2. Page titles (&lt;title&gt;)

Key pages use concise, keyword-forward titles so the SERP snippet (and sitelinks) stay readable:

- “Angel Number Meanings: Love, Career & Twin Flame”
- “Life Path Calculator | Spirit Numeral”
- “Compare Numbers | Spirit Numeral”

We avoid stuffing; we keep the main topic + optional modifier (e.g. “Love”, “Career”) in a short line.

### 3. Press kit

The **Compact Keywords** section on `/press` lists preferred short phrases for:

- **Headlines** when journalists or partners mention Spirit Numeral (e.g. “Angel Number Meanings”, “Life Path Calculator”)
- **Link text** when linking to us (e.g. “Numerology Quiz”, “Compare Numbers”)

Using the same phrases on our site and in external coverage reinforces what we’re about and can support sitelinks and rich results.

### 4. Internal links

When we link between our own pages (e.g. from blog posts or meaning pages), we use **concise anchor text** that matches the target page’s topic (e.g. “angel number 111”, “life path calculator”, “compatibility calculator”) instead of generic “click here” or long sentences.

---

## What we don’t do

- We don’t stuff long keyword lists into one title or one nav label.
- We don’t use different wording for the same page in nav vs footer vs press; we keep one compact phrase per destination.
- We don’t rely on “compact keywords” as a separate GSC metric; it’s a content and UX practice that supports sitelinks and clarity.

---

## Summary

| Area | Application |
|------|-------------|
| **Nav / footer** | Short labels: Angel Numbers, Life Path Calculator, Compare Numbers, etc. |
| **&lt;title&gt;** | Concise, topic + optional modifier; avoid long strings. |
| **Press kit** | Compact Keywords + Preferred Keywords sections; copy-ready phrases. |
| **Internal links** | Relevant, concise anchor text that describes the target page. |

This keeps our site and brand easy to describe and helps search engines show the right sitelinks and snippets.
